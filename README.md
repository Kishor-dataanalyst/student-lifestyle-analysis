# Impact of Daily Lifestyle Habits on Academic Performance

## Problem Statement
This project analyzes how daily lifestyle habits such as sleep,
study hours, stress, and physical activity impact academic performance.

## Dataset
The dataset contains 2000 student records with lifestyle and academic variables.
Source: Publicly available dataset from Kaggle.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Insights
- Adequate sleep is associated with higher CGPA
- High stress levels negatively impact academic performance
- Balanced study and sleep routines perform better than excessive studying
- Physical activity helps reduce stress levels


The analysis highlights the importance of maintaining a balanced lifestyle
for better academic outcomes.
